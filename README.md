# 7-Minute Workout App 💪
A fully functional Android application for 7-minute workout plans. The app integrates multiple features including user authentication, API data fetching, offline storage, notes management, and SQLite database persistence. It is built using Java and XML in Android Studio.

## 🚀 Key Features
- User authentication system (login, registration, and password reset)
- Landing screen with navigation to different modules
- Fetch workout data from a public API and display it using RecyclerView
- Offline storage of API data using SQLite database
- Notes feature with add and delete functionality
- SQLite-based local database for persistent note storage
- Fully responsive and user-friendly UI with proper spacing and layout design
- Error handling with offline fallback support
- Smooth navigation between multiple screens using Intents
- 7-minute workout routine consisting of 12 structured exercises with timed intervals
- Individual timers for each exercise with pause and resume controls
- Option to skip exercises and move directly to the next exercise
- Smooth and structured flow between exercises for better user experience

## 🛠️ Tech Stack
- Java
- XML (UI Design)
- Android Studio
- SQLite Database
- REST API Integration

## 📱 Description
This application is designed to provide a simple and efficient 7-minute workout experience while demonstrating core Android development concepts such as authentication, API integration, local data storage, and multi-screen navigation. The workout module is structured into 12 timed exercises, allowing users to pause, resume, or skip exercises based on their preference, ensuring flexibility and a smooth fitness experience.
