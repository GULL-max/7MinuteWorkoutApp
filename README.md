A fully functional Android app for 7-minute workout plans. The app integrates multiple features including user authentication, API data fetching, offline storage, notes management, theme switching, and SQLite database persistence. It is built in Java and XML using Android Studio. 

Key Features Implemented: 

User authentication (login, registration, password reset). 

Landing screen with navigation to other modules. 

Fetch workout data from a public API and display in a RecyclerView. 

Offline storage of API data using SQLite. 

Notes feature with add, save, and retrieve functionality. 

Persistent theme switching (Light/Dark). 

SQLite database for storing notes. 

Fully responsive UI with clean layout and proper padding/margins. 

Error handling and offline fallback. 

Navigation between multiple screens using Intents.
